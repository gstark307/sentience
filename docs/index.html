Source Code

Check it out

The source code can be downloaded using the Subversion version control system.  I'd recommend using a graphical client such as RapidSVN.

Guest
svn checkout http://sentience.googlecode.com/svn/trunk/ sentience-read-only

Project members
svn checkout https://sentience.googlecode.com/svn/trunk/ sentience --username <MyGoogleUsername>

or just browse the latest code


Source docs

Sentience consists of a number of separate projects which perform different tasks.  Documentation for each of these can be accessed below.  If you have the source code checked out the latest documentation may be automatically generated.  To do this make sure that doxygen and graphviz are installed, then run the create_documentation shell script from within the docs directory.
